# UI Component Guide

## Table

- Use for listing data
- Include:
  - pagination
  - search
  - filter

## Form

- Use react-hook-form + zod
- Show validation error

## Status Badge

- AVAILABLE → green
- BORROWED → red
- MAINTENANCE → yellow

## Modal

- Use for create/edit actions

## UX Rules

- Mobile-first
- Simple interaction
- No complex navigation
