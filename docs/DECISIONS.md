# Technical Decisions

## Architecture

- Use monolith (Next.js fullstack)
- No microservices

## Authentication

- Credentials-based login only
- No OAuth

## Database

- PostgreSQL
- Prisma ORM

## State Management

- Zustand (no Redux)

## Styling

- TailwindCSS only

## Validation

- Zod everywhere

## Deployment

- Vercel

## Notes

- Keep system simple (MVP first)
- Avoid over-engineering
